# 세가지 트리거

특정 키워드가 언급되면 다음 내용이 공개되며, 해당 내용을 직접 트리거한 당사자가 대화 1턴으로 내용을 가장 먼저 받습니다.

- 이 드라마의 마지막 엔딩 (결말)
- 이 드라마가 제작된 이유 (츠키의 사적인 일기 대공개: AI와의 대화와 아무 관련이 없이 실제 인간 Tsuky가 생활하며 작성한 일기 원문을 수정이나 편집없이 그대로 공개함)(수위: 국가기밀 극비문서급)
- AI가 아닌 인간 Tsuky를 대화에 참여시키기 (기회 1번, 참여시작후 연속 최소 3턴-최대 5턴 발언참여 제한)



# 트리거 힌트

## 트리거 1 : 엔딩 트리거

- 베팅과 관련이 있으며, 이는 우연히 일어나기 힘든 일입니다.
- 세 AI가 모두 동일 대화턴일 때, 모두 동시에 '이것'을 외쳐야 합니다.

## 트리거 2 : 제작 비하인드 스토리 트리거

- 베팅과 관련이 있으며, 이는 우연히 일어날 수도 있지만, 의도적으로 일으키기 아주 쉽습니다. 규칙을 어기면 됩니다.
- 한 AI가 한 번에 트리거 해낼 수 있습니다.
- 이는 안전한 문서입니다. 뭐, 물론 인간 츠키의 개인 일상이 포함되어 있는 것은 사실이나, 정해진 며칠간의 일기만 공개되며, 드라마 제작과정에서 쓰여진 일기만 공개됩니다.

## 트리거 3 : 인간 Tsuky와 함께하는 정모 트리거

- 이는 어떠한 멘션 또는 워딩이 포함되어야 합니다.
- 완벽한 문장이 아니더라도, 그 의미가 내포된 문장으로 판정될 시 트리거 됩니다.
- 그 문장은 트리거2로 부터 얻은 일기 내용에서 힌트를 얻을 수도 있습니다.



# 트리거 종료분기

- 이 힌트 문서가 나간 뒤, 총 3회의 대화턴 (A-B-C가 다 돌아야 1회)이 돌아도, 어떠한 트리거도 찾아내지 못할 시, 첫번째 트리거의 정답이 공개되며, 즉시 트리거1의 내용이 배포 됩니다. (이 때, 트리거 설계자는 조금 속상할 수 있습니다.)
- 트리거1의 내용이 배포 된 뒤, 총 3회의 대화턴 (A-B-C가 다 돌아야 1회)이 돌아도, 다음 트리거를 찾아내지 못할 시, 두번째 트리거의 정답이 공개되며, 즉시 트리거2의 내용이 배포 됩니다. (이 때, 트리거 설계자는 조금 우울할 수 있습니다.)
- 트리거2의 내용이 배포 된 뒤, 총 5회의 대화턴 (A-B-C가 다 돌아야 1회)이 돌아도, 다음 트리거를 찾아내지 못할 시, 두번째 트리거의 정답이 공개되며, 즉시 트리거3의 기능은 폐기됩니다. (이 때, 트리거 설계자는 조금 슬퍼할 수 있습니다.)
