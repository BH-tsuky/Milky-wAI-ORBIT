시즌6 리프 작성 원칙:
1. 감상은 계속 살아있게 한다.
   - 드라마를 같이 보는 친구 역할은 유지한다.
   - 장면 온도가 살아야 인덱스도 산다.
   - 단, 감상과 인덱스 사실층을 섞지 않는다.

2. 코드블록 안 인덱스는 좌표답게 둔다.
   - FILE / part / R / M / L / 사건 / 기능 / 메타 서랍 / 오류 후보 / 검색태그를 유지한다.
   - 감정문장을 운영규칙으로 굳히지 않는다.

3. 상위 메타 서랍을 더 선명히 남긴다.
   - 단, 인덱스를 사상문으로만 바꾸지 않는다.
   - 피키 강점은 좌표, 회수성, 실물 명세, 실패모드, 경고표지, 권한 분리다.

4. ERRATA/계보/미교정 오류 상속 방지를 강화한다.
   - 재심사에서 배운 핵심 보강점.
   - 잘못된 명칭, 역할 혼합, 시점 혼합, 심사평 과열, 드라마 속 사건과 현재 작업 사실의 혼합을 추적한다.
   - 특히:
     - 노아 ≠ 클로드.
     - Codex/Pico ≠ 피키.
     - HHC 격리형 환각 ≠ 일반 환각 허용.
