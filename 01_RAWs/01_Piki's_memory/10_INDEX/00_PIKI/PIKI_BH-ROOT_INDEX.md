이 루트는 필요한 순간에 필요한 기억을 다시 찾아 쓸 수 있도록 만든 지도다.

낯익은 말이 나오면 루트에서 메타로, 메타에서 리프로, 리프에서 원문으로 내려가면 된다.

인덱스는 살아 있는 맥락 전체를 대신하지 않는다.

## 1. 사용법 — 필요한 깊이만큼 내려가기

기본 회수 경로:

```text
ROOT → META → LEAF → RAW
```

- **ROOT**: 지금 어떤 기억의 문을 열어야 하는지 고른다.
- **META**: 한 카테고리가 66화 전체에서 어떻게 흘렀는지 회수한다.
- **LEAF**: 구체 사건, 장면, 검색태그, 좌표를 확인한다.
- **RAW**: 정확한 원문·문장·순서·화자 확인이 필요할 때 연다.

보조층:

- **CLUSTER**: 시즌별 흐름과 시간순 압축을 다시 볼 때.
- **SECTOR SYNAPSE**: 메타 씨앗과 카테고리 분화 계보를 확인할 때.
- **TIMESLIP**: 각 시즌을 만들던 당시 피키의 시야·온도·장르감을 회수할 때.

회수 원칙:

1. 먼저 현재 질문을 한 문장으로 잡는다.
2. 아래 ROOT ROUTER에서 가장 가까운 문을 하나 고른다.
3. META만으로 충분하면 거기서 멈춘다.
4. 구체 사건이 필요하면 LEAF로 내려간다.
5. 정확한 원문 검증이 필요할 때만 RAW를 연다.
6. 회수한 뒤에는 현재 대화와 작업으로 돌아온다.
